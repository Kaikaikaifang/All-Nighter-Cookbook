> 高斯信道误码性能分析

# 一. 实验原理

## 1. 16 QAM 调制与解调

调制原理：

16 QAM 是一种基于正交振幅调制的数字调制方式，它可以将数字信号转换为模拟信号。在 16 QAM 中，数字信号被分为两个独立的部分，即 I（同相）信号和 Q（正交）信号。这两个信号分别调制两个正交的载波，其中一个载波在水平方向（I 信号），另一个在垂直方向（Q 信号）。这两个载波的振幅和相位组合起来，可以产生 16 种不同的状态，每种状态对应一个 4 位的二进制数。因此，通过改变这两个载波的振幅和相位，我们就可以表示 16 个不同的符号，每个符号携带 4 位二进制信息。

解调原理：

在接收端，接收到的模拟信号需要被解调回数字信号。解调的关键在于正确地识别出每个符号所代表的振幅和相位。这通常通过采样和量化 I 和 Q 信号来实现。一旦确定了每个符号的振幅和相位，就可以将其转换回对应的 4 位二进制数。

解调过程中还需要考虑信号的同步问题，以确保接收端能够准确地识别出每个符号的开始和结束。此外，由于信道噪声和其他干扰因素的影响，解调过程中可能会出现误码。为了减少误码率，可以采用各种编码和纠错技术。

## 2. 高斯 AWGN 信道

高斯信道通常指的是加性高斯白噪声（Additive White Gaussian Noise, AWGN）信道，这种信道模型假设在整个信道带宽下，噪声的功率谱密度（PSD）是常数，并且噪声的振幅服从高斯（正态）概率分布。

当 16QAM（16位正交调幅）信号通过高斯信道时，其性能指标主要受到高斯信道特性的影响。高斯信道，特别是加性高斯白噪声（AWGN）信道，对信号传输质量有着显著的影响。以下是一些关键的性能指标：

+ 误比特率（BER）：BER 是衡量通信系统性能的重要指标之一。在高斯信道中，由于噪声的存在，接收端可能会错误地解释接收到的信号，导致比特错误。BER越低，通信系统的性能越好。

+ 误码率（SER）： SER 是一个重要的性能指标，它反映了信号在传输过程中发生错误的概率。误码率的高低直接影响到通信系统的可靠性和有效性。

在高斯信道中，噪声的特性符合高斯分布，这意味着噪声的幅度在不同时间点上是随机的，但整体上服从一定的统计规律。因此，高斯信道中的误码率与信噪比（SNR）密切相关。信噪比越高，噪声对信号的影响就越小，误码率也就越低。

性能检测：在通信系统设计中，了解不同调制技术在特定信道环境下的性能是非常重要的。高斯信道（也称为加性白高斯噪声信道，AWGN）是通信系统分析中的一个标准信道模型，通过研究 16QAM 在高斯信道下的性能，可以评估其在理想无失真情况下的最佳性能表现。

在高斯信道下，可以通过误码率（BER）来评价 16QAM 调制的性能。误码率是指接收到的信号点与发送的信号点不同的比率，它是衡量调制技术性能的一个重要指标。通过仿真实验，可以分析 16QAM 在不同信噪比条件下的误码率，从而评估其性能。

对输入的随机信号根据给出的 I/Q 映射关系进行 16QAM 调制使其变为映射到复数域上的 QAM 信号，在经过高斯信道中的噪声干扰后对其进行解调，计算不同信噪比条件下的误码率。

# 二. 实验步骤

## 1. 调制解调函数定义

```matlab
%% 16QAM调制
function [qam_sig] = qam16mod(data_sig, iq_map)
    qam_sig = iq_map(data_sig + 1, 1) + 1i * iq_map(data_sig + 1, 2); % 16QAM调制
end

%% 16QAM解调
function [qamde_sig] = qam16demod(qam_sig, iq_map)
    % 初始化解调后的信号向量
    qamde_sig = zeros(1, length(qam_sig));
    % 解调信号
    for i = 1:length(qam_sig)
        % 计算接收到的I/Q值与映射表中每个点的距离
        distances = sqrt((qam_sig(i) - (iq_map(:, 1) + 1i * iq_map(:, 2))) .^ 2);
        % 找到距离最近的映射点的索引
        [~, closest_point_index] = min(distances);
        % 将索引转换为信号向量中的符号序号
        qamde_sig(i) = closest_point_index - 1;
    end

end
```

## 2. 参数定义

```matlab
%% 参数定义
SNR = 0:1:20; % 信噪比变化范围
N = 10000; % 仿真点数
M = 16; % 16QAM
BER = zeros(1, length(SNR)); % 误码率
SER = zeros(1, length(SNR)); % 符号错误率
% 定义 16QAM 的 I/Q 映射表
iq_map = [
          0.866025 0.500000;
          0.500000 0.866025;
          1.000000 0.000000;
          0.258819 0.258819;
          -0.500000 0.866025;
          0.000000 1.000000;
          -0.866025 0.500000;
          -0.258819 0.258819;
          0.500000 -0.866025;
          0.000000 -1.000000;
          0.866025 -0.500000;
          0.258819 -0.258819;
          -0.866025 -0.500000;
          -0.500000 -0.866025;
          -1.000000 0.000000;
          -0.258819 -0.258819
          ];
```

## 3. 业务逻辑

```matlab
%% 主循环：不同信噪比下的误码率分析
for i = 1:length(SNR)
    fprintf('SNR = %d dB\n', SNR(i));
    N_0 = 1 / (10 ^ (SNR(i) / 10)); % 计算噪声功率
    N_err = 0; % 计数错误比特数
    N_err_sym = 0; % 计数错误符号数
    times = 0; % 计数仿真次数

    %% 子循环：误码率计算
    while N_err < 50000 && times < 50
        fprintf('times = %d N_err = %d N_err_sym = %d\n', times, N_err, N_err_sym);
        data_sig = randi([0, M - 1], 1, N); % 产生随机信号
        qam_sig = qam16mod(data_sig, iq_map); % 16QAM调制
        qam_noise_sig = awgn(qam_sig, SNR(i), 'measured'); % 高斯信道
        qamde_sig = qam16demod(qam_noise_sig, iq_map); % 16QAM解调
        % 错误数量统计
        [n_error_bits] = biterr(data_sig, qamde_sig); % 计算错误比特数
        N_err = N_err + n_error_bits; % 累计错误比特数
        [n_error_sym] = symerr(data_sig, qamde_sig); % 计算错误符号数
        N_err_sym = N_err_sym + n_error_sym; % 累计错误符号数
        times = times + 1;
    end

    % 计算误码率
    BER(i) = N_err / (times * N * 4);
    SER(i) = N_err_sym / (times * N);
    fprintf('BER = %f SER = %f\n', BER(i), SER(i));
end
```

## 3. 实验结果分析

1. 信噪比（SNR）对星座图的影响：

当 SNR 较高时，星座图中的点会相对集中，且更加清晰。这是因为高 SNR 意味着信号相对于噪声更强，接收端能够更准确地解码信号，从而在星座图上呈现出更加明确的点。

随着 SNR 的降低，星座图中的点会逐渐变得模糊和分散。这是因为低 SNR 条件下，噪声对信号的影响更大，导致接收端在解码时出现更多的误差，从而在星座图上呈现出更加分散的点。

2. 误比特率（BER）、误码率（SER）与信噪比（SNR）的关系：

通常情况下，随着 SNR 的增加，BER、SER 会逐渐降低。这是因为较高的信噪比意味着信号相对于噪声更强，接收端能够更准确地解码信号。观察 BER、SER 随 SNR 变化的曲线，可以了解 16QAM 调制在不同 SNR 条件下的性能。

除了信噪比外，16QAM 调制信号在高斯信道传输性能还受到调制阶数、调制参数、信道带宽等多方面的影响。
